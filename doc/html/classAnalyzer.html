Detailed Description

Class for analyzing the data.

Creates the TChain and branch addresses for the passed tree.
Creates all the basic data variables.
ADC and TDC data are just done with simple variables.
Flash ADC data is handled with running TClonesArray on TArtFadcHit.

      //GEM-MSTPC
      if (flag_tpc){  // analyze TPC data?
        
        TArtFadcHit *fadc;

                      if (TMath::Abs(trough-baseline[id][fNHit[id]])>(baseline_dev[id][fNHit[id]]*baseline[id][fNHit[id]])){
